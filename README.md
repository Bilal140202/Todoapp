# Premium Notes App

A beautiful, premium-grade notes application built with Flutter featuring ultra-high quality UI/UX.

## Features

- Create, edit, and delete notes
- Rich text support
- Color-coded notes (14 beautiful colors)
- Pin important notes to top
- Favorite notes for quick access
- Archive notes
- Search functionality
- Smooth animations
- Dark/Light theme support
- Local SQLite database
- Trash with 30-day retention
- Beautiful Material 3 design

## Premium Features

- Staggered grid layout for notes
- Swipe actions (pin, delete)
- Category support
- Date formatting (relative time)
- Smooth page transitions
- Custom color picker
- Premium typography (Inter font)
- Beautiful empty states
- Haptic feedback ready

## Tech Stack

- Flutter 3.x
- Dart 3.x
- flutter_bloc for state management
- SQLite for local storage
- flutter_animate for animations
- flutter_staggered_grid_view for layout
- Google Fonts (Inter)

## Architecture

This app follows clean architecture principles:
- Feature-first organization
- MVVM pattern
- Repository pattern
- BLoC state management
- Separation of concerns

## Building

```bash
flutter pub get
flutter build apk --release
```
